# Project Overview

This project demonstrates the use of Python data structures including **Lists, Tuples, Sets, and Dictionaries**. It also combines these data structures to build a simple product catalog management system.

---

# Task 1: Product Collections (Lists & Tuples)

## Objective

Practice working with Python lists and tuples.

### Features

- Create a list named `products` containing at least six product names.
- Create a tuple named `sample_product` containing:
  - Product Name
  - Price
  - Category
- Display the second and last products from the list.
- Add two new products to the list.
- Print the updated product list.
- (Optional) Convert the tuple into a list, update the price, and convert it back into a tuple.

### Python Concepts Used

- Lists
- Tuples
- List Indexing
- List Methods (`append`)
- Type Conversion

# Task 2: Categories (Sets)

## Objective

Learn how Python sets store unique values.

### Features

- Create a set named `categories_set`.
- Add a new category.
- Demonstrate that duplicate categories are ignored.
- Check whether a category exists in the set.
- Display the total number of unique categories.

### Python Concepts Used

- Sets
- `add()`
- Membership Operator (`in`)
- `len()`

# Task 3: Product Pricing (Dictionaries)

## Objective

Manage product prices using Python dictionaries.

### Features

- Add a new product with its price.

- Remove a product safely.
- Calculate the average price of all products.
- Display the highest-priced product.
- Display the lowest-priced product.

### Python Concepts Used

- Dictionaries
- Dictionary Methods
- `max()`

- Loops

Combine Lists, Tuples, and Dictionaries into a simple product catalog.
