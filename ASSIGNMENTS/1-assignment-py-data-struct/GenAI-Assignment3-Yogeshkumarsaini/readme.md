# Project Description

This assignment demonstrates the use of Python functions to solve common programming problems. The project covers user-defined functions, recursive functions, lambda expressions, map(), filter(), and menu-driven programming using functions.

---

# Tasks Included

## Task 1 - Price After Discount

- Create a function `apply_discount(price, discount_percent)`
- Uses a default discount of 5%
- Calculates the final price after discount

---

## Task 2 - Recursive Factorial

- Create a recursive function `factorial(n)`
- Handles:
  - Positive numbers
  - 0
  - 1
  - Negative numbers

---

## Task 3 - Lambda Function (GST Calculator)

- Create a lambda function to calculate price after adding 18% GST.
- Optional lambda to calculate GST and discount together.

---

## Task 4 - Using map()

- Apply GST to a list of prices using `map()`.
- Display original and updated prices.

---

## Task 5 - Using filter()

- Filter prices greater than 500.
- Filter prices less than or equal to 500.

---

## Task 6 - Combined Utility Function

- Create a function `process_prices(prices)`
- Apply 10% discount using `map()`
- Filter prices greater than 300 using `filter()`
- Return both lists

---

## Task 7 - Menu Using Functions

Create three functions:

- add_price()
- get_average_price()
- get_max_price()

Menu Options:

- Add Price

- Show Highest Price
